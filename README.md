编号：001

功能名称：人物走动（无武器）

动画名称：IdleNonWeapon, RunNonWeapon

具体功能描述：
        AD分别控制人物左右移动，在攻击状态下无法移动，在换方向时人物自动更换脸的朝向，包含一个粒子系统（脚边的灰尘）。
        当人在非走动的状态下切换成待机状态。

功能相关animator参数：
        1.XSpeed：
                a.类型：float
                b.描述：人物在x轴方向的速度

功能相关代码及描述：（需要添加到gameObject上的代码以及为了实现该功能而修改过的代码，::表示继承）
        1.代码名称：MainCharacter.cs--Update(),move(),Flipping()[::Livings.cs]
                a.调用代码的gameObject名称：MainCharacter

功能实现细节：（涉及与代码与scene交互的步骤用*号标出）
        1.添加rigibody2D属性给MainCharacter
        *2.在代码中将Speed设置为人物速度
        3.建立从站立动画到行走动画的transition，转换条件为当Speed绝对值大于0.1时触发人物走动的动画，小于0.1时触发站立动画。
        4.人物在跑动的时候 P_Run (ParticalSystem)激活，静止不动时候 P_Run处于关闭状态。
        *5.脚边灰尘的粒子系统开关控制：当人只处于跑动状态时开启，其他状态均不开启。当人物反向的时候，粒子系统也需要反向。

功能涉及参数及描述：
        1.movespeed:
                a.类型：float
                b.描述：移动速度，可修改。

编号：002

功能名称：人物跳跃（无武器）

动画名称：JumpNonWeapon

具体功能描述：
        Space控制人物的跳跃，暂时不可以二段跳，无粒子系统。

功能相关animator参数：
        1.Grounded:
                a.类型：Bool
                b.描述：检测人是否站在地上

功能相关代码及描述：
        1.代码名称：MainCharacter.cs--Update(),FixedUpdate()
                a.调用代码的gameObject名称：MainCharacter

功能实现细节：
        1.创建一个空的GameObject，名称为CheckGround
        2.将CheckGround拖到人物脚下
        3.将所有的地面元素添加到Ground层
        4.在MainCharacter的MainCharacter属性中，将CheckGround拖到groundCheck项中，设置groundCheckRadius（0.3左右可调），将whatIsGround设置为Ground
        *5.检测以groundCheck位置为圆心，groundCheckRadius为半径的圆是否与whatIsGround相交，如果相交Grounded为true，反之为false
        6.CheckGround为false时触发跳跃动画，反之切换到站立动画

功能涉及参数及描述：
        1.groundCheck
                a.类型：Transform
                b.描述：对应场景中的CheckGround的Transform
        2.groundCheckRadius
                a.类型：float
                b.描述：以groundCheck位置为圆心，groundCheckRadius为的圆的半径，当这个圆与地面相交时Grounded为true，反之false
        3.whatIsGround
                a.类型：LayerMask
                b.描述：对应场景中所有位于Ground层的元素

动画名称：HitNonWeapon

具体功能描述：
        在人物受到敌方单位普通攻击时，获得短暂时间的无敌时间，不会被攻击。在此期间不能进行攻击、行走、跳跃、释放攻击技能。
        可以向前向后翻滚，可以释放防御技能。

功能相关animator参数：
        1.BeAttacked:
                a.类型：Trigger
                b.描述：当受到攻击时，触发该trigger，播放被攻击动画。无敌时间等于该动画长度

功能相关代码及描述：
        1.代码名称：Minion.cs::Livings.cs(Livings.cs中发生了修改，该修改通过继承影响Minion)--DefaultAttack()
                a.GameObject名称：MinionFrog
        2.代码名称：MainCharacter.cs--Update(),BeAttacked()
                a.GameObject名称：MainCharacter

功能实现细节：
        *1.受到攻击时，触发BeAttacked
        2.调用相应动画
